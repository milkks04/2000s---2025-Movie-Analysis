Overview

This project focuses on movies released between 2000 and 2025. The main goal is to track how movie quality and output changed as the industry moved from a theater-first model to a streaming-dominated world.

The analysis cleans the dataset, extracts the release year, and uses grouping, summary statistics, and visualizations to compare different time periods.

  • Release year extracted from full release dates.
  • Filtering to just 2000‚Äì2025 for a modern focus.
  • Year-level summaries for ratings, counts, votes & popularity.
  • HTML ‚ÄúINFO‚Äù blocks below each plot explain what the chart shows.

Visuals

The full notebook includes multiple visualizations with explanations. A few highlights:

  • Average Rating by Year: shows ratings staying surprisingly stable.
  • Movies per Year: reveals how the 2020s explode in volume.
  • High-Rated Heatmap: shows when and in which languages 8.0+ films appear.
